## 표기 범례

```
[ 버튼 텍스트 ]   : 클릭 가능한 버튼
[ 입력 필드   ]   : 텍스트 입력 필드
[ 드롭다운   ▼]   : 셀렉트 박스
( ○ ) / ( ● )    : 라디오 버튼 (미선택 / 선택)
[ ] / [✓]         : 체크박스 (미선택 / 선택)
~~텍스트~~         : 취소선 (완료된 할일)
! 빨간색 경고      : 에러 메시지 / 기간 초과 강조
→ 이동             : 화면 전환 방향
```

## 공통 컴포넌트

### 헤더 (인증 후)

```
+----------------------------------------------+
| TodoListApp              김지우 님  [로그아웃] |
+----------------------------------------------+
```

### 할일 카드 (기본 / 완료 / 기간 초과)

```
[ 미완료 카드 ]
+------------------------------------------+
| [ ] 마케팅 전략서 작성                     |
|     [업무]  📅 2026-05-20          [수정] |
+------------------------------------------+

[ 완료된 카드 ]
+------------------------------------------+
| [✓] ~~장보기 목록 정리~~                   |
|     [쇼핑]  📅 2026-05-10          [수정] |
+------------------------------------------+

[ 기간 초과 카드 (붉은색 강조) ]
+------------------------------------------+
| [ ] 선형대수 5장 연습문제                  |
|     [개인]  ⚠ 2026-05-08 기간초과  [수정] |
+------------------------------------------+
```

### 인라인 에러 메시지

```
  이메일
  +--------------------------------------+
  | invalid-email                        |
  +--------------------------------------+
  ! 올바른 이메일 형식을 입력하세요.
```

### 토스트 알림 (API 오류)

```
+---------------------------------------------------+
| ✕  이메일 또는 비밀번호가 잘못되었습니다.    [×] |
+---------------------------------------------------+
  ↑ 화면 상단 고정. 3초 후 자동 닫힘
```

### 빈 목록 상태

```
+------------------------------------------+
|                                          |
|            할일이 없습니다.              |
|       첫 번째 할일을 추가해보세요.       |
|                                          |
|          [ + 할일 추가하기 ]             |
|                                          |
+------------------------------------------+
```

| SCR-03 | [+ 할일 추가] 클릭 | `/todos/create` 이동 |
| SCR-03 | [로그아웃] 클릭 | Zustand 스토어 초기화 → `/login` 이동 |
| SCR-04 | [저장] 클릭 | 유효성 검사 → `POST /api/todos` → `/todos` 이동 |
| SCR-04 | [취소] 클릭 | `/todos` 이동 (변경사항 미저장) |
| SCR-05 | [저장] 클릭 | 유효성 검사 → `PUT /api/todos/:id` → `/todos` 이동 |
